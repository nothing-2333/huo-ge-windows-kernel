# 查看寄存器
r 寄存器

# 查看地址的内存，（物理内存要在最前面加！，虚拟内存不用）
## 注意显示出来的内存全是小端序
db/dw/dd/dq 寄存器/address [l 0x20]
## 竖向查看
dds/dqs 寄存器/address [l 0x20]
## 查看物理内存
!dd address
## 显示内存的unicode
!du address

# 改变内存（物理内存要在最前面加！，虚拟内存不用）
eb/ew/ed/eq address target
